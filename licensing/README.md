# Licensing

Модуль защиты лицензий для Aegis-Q.

## Функциональность

- Обфускация ключей
- Защищённая конфигурация
- Aegis-Q envelope для передачи лицензий
- Защита бинарей (встраиваемый модуль)

## Использование

```rust
use licensing::{License, ObfuscatedKey, ProtectedConfig, LicenseEnvelope};
```
